# Assistente Virtual em Python

## Descrição
Projeto de um assistente virtual capaz de ouvir comandos de voz e executar ações como:

- Pesquisar na Wikipedia
- Abrir YouTube
- Mostrar farmácia próxima
- Tocar músicas

O assistente também responde em áudio.

## Tecnologias
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia
- PyWhatKit
- Webbrowser

## Como usar
1. Instalar dependências:
pip install SpeechRecognition pyttsx3 wikipedia pywhatkit pyaudio
2. Executar:
(Exemplos de comandos)
- "abrir youtube"  
- "pesquisar wikipedia inteligência artificial"  
- "toque lo-fi"  
- "farmácia"  
- "encerrar"
